Triply Travel Planner

Triply is an Android mobile application that helps users plan vacations, organize excursions, manage travel details, and validate trip dates in one place.

Features

Triply allows users to create, update, and delete vacation records.

Users can add excursions to each vacation and organize travel details clearly.

The application validates excursion dates so they stay within the vacation date range.

Triply includes a user-friendly interface designed for simple travel management.

The app was developed as part of a software development capstone project.
